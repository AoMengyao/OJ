﻿﻿### NC1 大数加法

> <font color='black'>**描述**</font>
>
> 以字符串的形式读入两个数字，编写一个函数计算它们的和，以字符串形式返回。
>
> 数据范围：s.length,t.length<=100000，字符串仅由'0'~‘9’构成
>
> 要求：时间复杂度 O(n)*O*(*n*)

><font color='black'>**示例1**</font>
>
>```
>输入："1","99"
>返回值："100"
>说明：1+99=100 
>```
>
><font color='black'>**示例2**</font>
>
>```
>输入："114514",""
>返回值："114514"
>```
